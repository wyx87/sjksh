# 数据可视化网页

基于Matplotlib示例的交互式数据可视化仪表板。

## 项目描述

本项目将Jupyter笔记本中的Matplotlib数据可视化示例转换为交互式网页，使用Chart.js实现。

## 功能特性

- 🌡️ 温度折线图（最高/最低气温）
- 💰 阿里巴巴GMV柱形图
- 🛒 网购替代率条形图
- 💳 支付宝账单饼图
- 🚗 汽车速度散点图
- 🎯 霍兰德职业测试雷达图
- ⚡ 发电量箱形图
- 📦 物流费用面积图

## 使用方法

1. 直接打开 `index.html` 文件在浏览器中查看
2. 或使用本地服务器：
   ```bash
   python -m http.server 8080
   ```
   然后访问 http://localhost:8080

## 技术栈

- HTML5
- CSS3 (响应式设计)
- Chart.js (数据可视化)
- JavaScript (交互功能)

## 数据来源

数据来源于Matplotlib学习示例，包含多种真实场景的数据可视化案例。
