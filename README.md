# Matplotlib第2章 数据可视化仪表板

基于Matplotlib第2章内容开发的专业数据可视化网页应用，包含10种经典图表类型的交互式展示。

## 🎯 项目概述

本项目是一个功能丰富的数据可视化仪表板，完全基于Matplotlib第2章的教学内容，涵盖了数据可视化的核心图表类型。通过现代化的Web技术实现，提供了优雅的用户界面和强大的交互功能。

## 📊 包含的图表类型

### 2.1 折线图
- **示例**：未来15天最高气温和最低气温
- **特点**：双折线对比，带有数据点标记

### 2.2 柱形图  
- **示例**：2013—2019财年阿里巴巴淘宝天猫GMV
- **特点**：彩色柱形图，数据标签显示

### 2.3 条形图
- **示例**：各商品种类的网购替代率
- **特点**：水平条形图，渐变色彩效果

### 2.4 堆积面积图
- **示例**：物流公司物流费用统计
- **特点**：多层数据堆积，面积填充效果

### 2.5 直方图
- **示例**：人脸识别的灰度直方图
- **特点**：10000个数据点的分布展示

### 2.6 饼图
- **示例**：支付宝月账单报告
- **特点**：圆环图样式，百分比显示

### 2.7 散点图
- **示例**：汽车速度与制动距离关系
- **特点**：数据点连线，趋势展示

### 2.8 箱形图
- **示例**：2017年和2018年全国发电量统计
- **特点**：箱线图对比，统计信息展示

### 2.9 雷达图
- **示例**：霍兰德职业兴趣测试
- **特点**：多维度数据对比，六边形雷达

### 2.10 误差棒图
- **示例**：4个树种不同季节的细根生物量
- **特点**：柱形图+误差棒，数据不确定性展示

## 🎨 主要功能

### 🌈 主题系统
- **8种淡色主题**：浅色、奶油、薄荷、薰衣草、蜜桃、天空、玫瑰、鼠尾草
- **实时切换**：主题切换立即生效
- **持久化**：自动保存用户主题偏好

### 📱 视图模式
- **总览模式**：同时显示所有10个图表
- **分页模式**：分页浏览，每页6个图表
- **页面导航**：上一页/下一页快速切换

### ✏️ 数据编辑
- **实时编辑**：在线修改图表数据
- **即时更新**：数据改变立即反映在图表上
- **多数据支持**：支持多个图表的数据独立编辑

### 🎲 数据管理
- **随机数据生成**：一键生成测试数据
- **数据重置**：恢复到默认教学数据
- **数据统计**：实时显示数据点数量

### 🖼️ 图表交互
- **双击全览**：双击任意图表打开全屏大图
- **ESC关闭**：按ESC键快速关闭全览
- **鼠标悬停**：显示详细数据信息

### 💾 导出功能
- **批量导出**：一键导出所有图表为PNG图片
- **高清质量**：1200x800分辨率
- **自动命名**：包含时间戳的文件名

### 🖨️ 打印功能
- **专业布局**：优化的打印样式
- **分页处理**：避免图表跨页显示
- **隐藏控件**：打印时自动隐藏控制元素

## 🚀 使用方法

### 1. 基本使用
1. 双击`index.html`文件在浏览器中打开
2. 浏览所有10种图表类型
3. 使用右上角切换器选择喜欢的主题

### 2. 高级功能
1. **数据编辑**：点击"数据编辑"按钮，修改文本框中的数据
2. **分页浏览**：点击"分页模式"，使用导航按钮翻页
3. **全览查看**：双击任意图表查看全屏大图
4. **导出图表**：点击"导出图表"保存所有图表

### 3. 快捷键
- **ESC**：关闭全屏图表
- **F5**：刷新页面重置数据
